# Key Features

- UI built with Vaadin Framework 8  
- View management with a good menu  
- Everything is built mobile first.  
- Software packaged with Spring Boot  
- Database connection. In-memory or file based h2 when developing and a traditional SQL database in production  
- Backend layer built with Spring Data  
- Master-detail view  
- Login with permissions and user management, built with Spring Security. This will be the base for our Vaadin Login product  
- Views built with Designer  
- Unit tests built with JUnit and integration tests with Testbench. One screenshot smoke test.  
- Dashboard built with Vaadin Board and Vaadin Charts  
- Instructions on each part of the showcased best practices, how to use them and how to switch out parts.  
- Build scalability tests with Testbench and gatling and write a report on performance, including for example amount of simultaneous users that the system can handle. (Gatling)  
- Instructions which show how to do scalability to push the user limit even higher. No code included in product itself.
